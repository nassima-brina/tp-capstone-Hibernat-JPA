# Application Réservation de Salles
## Description :
#### Ce projet est une application Java développée avec Hibernate et JPA permettant de gérer les salles, les utilisateurs, les équipements et les réservations dans une organisation.L'application permet de rechercher des salles disponibles selon plusieurs critères, de gérer les réservations et d'analyser les performances des requêtes Hibernate.Ce projet a été réalisé dans le cadre d'un TP Capstone sur Hibernate & JPA afin de démontrer l'utilisation des fonctionnalités avancées de Hibernate comme le cache de second niveau, la pagination, les requêtes optimisées et l'optimistic locking
## Technologies utilisées :
#### - Java
#### - Hibernate ORM
#### - JPA (Java Persistence API)
#### - H2 Database
#### - Maven

## Conclusion :
#### Ce projet a permis d'appliquer les concepts de Hibernate et JPA pour développer un système de gestion des réservations de salles. Les tests réalisés ont permis d'analyser les performances des requêtes et d'utiliser des techniques d'optimisation comme la pagination, JOIN FETCH et le cache.Ainsi, ce projet montre l'importance d'une bonne gestion de la persistance des données afin d'améliorer les performances et la fiabilité d'une application.
